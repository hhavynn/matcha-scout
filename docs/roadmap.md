# Roadmap

## Short-Term Improvements

- Enable Gemini in production after rate limits, monitoring, and cost controls are stable.
- Add real user feedback collection for recommendation quality.
- Improve search and filter refinements on the browse page.
- Add basic analytics for quiz submissions, popular filters, and empty-result cases.
- Add a short demo GIF or refreshed screenshot set for the README.

## Medium-Term Improvements

- Allow user-submitted cafes and drinks.
- Add a moderation queue for submitted cafes, drinks, and reviews.
- Add rate limiting and spam protection for review submission.
- Add automated preview deployment checks to the existing test/build CI.
- Tighten production access and document least-privilege policies.

## Out Of Scope For Now

- Production Kubernetes or EKS.
- Payments, accounts, or social features.
- Large-scale scraping of real cafe menus.
- Complex geospatial search.
